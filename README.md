# monit_aer_GE1.5
Sistema de captura, procesamiento, visualización y análisis de datos de un aerogenerador GE 1.5. El proyecto obtiene datos del PLC, procesa las señales recibidas, las almacena en CSV, las expone mediante una API, las visualiza en un dashboard con Streamlit y aplica un modelo Isolation Forest para detectar comportamientos anómalos.
